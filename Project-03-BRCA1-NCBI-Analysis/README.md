# BRCA1 Gene Analysis using NCBI Databases

## Project Overview

This project demonstrates the complete bioinformatics analysis of the human BRCA1 gene using various NCBI databases and tools. The analysis includes gene identification, nucleotide sequence retrieval, protein analysis, BLAST sequence validation, Gene Ontology annotation, and clinical variant interpretation.

---

## Objective

- Analyze the BRCA1 gene using NCBI resources.
- Retrieve nucleotide and protein sequences.
- Validate the sequence using BLAST.
- Study Gene Ontology annotations.
- Explore disease-associated variants using ClinVar.

---

## Databases Used

- NCBI Gene
- NCBI Nucleotide
- NCBI Protein
- BLAST
- ClinVar
- Gene Ontology

---

## Workflow

1. BRCA1 Gene Search
2. Transcript Analysis
3. FASTA Retrieval
4. Protein Analysis
5. Genome Browser Analysis
6. ClinVar Variant Analysis
7. BLAST Validation
8. Gene Ontology Analysis
