# Python-Based Student Management System

This project demonstrates CRUD (Create, Read, Update, Delete) operations using Python.

## Features

- Add Student
- View Student
- Update Student
- Delete Student
- Search Student

## Technologies Used

- Python
- Functions
- Loops
- Conditional Statements
- Dictionaries

## Learning Outcome

This project helped me understand Python fundamentals and CRUD operations while developing a simple console-based application.
