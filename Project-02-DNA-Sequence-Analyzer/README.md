# DNA Sequence Analyzer Using Python

## 📌 Project Overview

The DNA Sequence Analyzer is a Python-based bioinformatics project developed to perform basic biological sequence analysis. This project demonstrates how Python programming can be applied to solve fundamental bioinformatics problems involving DNA sequences.

The application performs sequence validation, nucleotide counting, FASTA file processing, and RNA transcription. It was developed progressively through multiple versions and finally integrated into a complete DNA sequence analysis tool.

---

## 🎯 Objectives

- Learn biological sequence analysis using Python
- Validate DNA sequences
- Count nucleotide frequencies
- Read FASTA files
- Perform RNA transcription
- Build a foundation for advanced bioinformatics programming

---

## 🛠 Technologies Used

- Python 3

- FASTA Format
- File Handling
- String Manipulation

## 📂 Project Features

- FASTA File Reading
